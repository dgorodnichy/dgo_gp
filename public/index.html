Anonymous block argument in Ruby

In the Ruby programming language, it is possible to use default block parameters instead of variables. This feature allows for more concise and expressive code. Before Ruby 2.7 We would need to write code like this: irb(main):005> [1, 2, 3].map { |el| el } => [1, 2, 3] From Ruby 2.7 However, in Ruby 2.7, numbered parameters were introduced, which allows us to write code like this: irb(main):010> [10, 100, 1000].
